      Hur SGCs XP system fungerar
      
    
        
  • XP tjänas genom att skicka meddelanden på servern.
    • 
        
  • Du kan bara få XP ungefär en gång var tionde minut.
    • 
        
  • Vanliga meddelanden ger vanligtvis 3 XP.
    • 
        
  • Upprepade/skräpmeddelanden ger bara 1 XP.
    • 
        
  • Meddelanden som skickas innan nedkylningstiden är slut ger 0 XP.
    • 
        
  • XP som krävs för att gå upp i nivå baseras på: level2.
    • 
        
  • När man går upp i nivå återställs XP till 0.
    • 
      

      

        Exempel:

        Nivå 10 → 100 XP behövs

        Nivå 20 → 400 XP behövs
